🏠 House Price Prediction (Multiple Linear Regression)
📌 Overview

This project predicts house prices using Multiple Linear Regression based on features like square footage, bedrooms, bathrooms, lot size, garage size, year built, and neighborhood quality.

📊 Dataset

The dataset contains 1000 records with 7 input features and one target variable (House_Price).

⚙️ Tools & Technologies

Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

🔄 Workflow
Loaded and explored dataset
Performed EDA and visualization
Split data into training and testing sets
Trained Multiple Linear Regression model
Evaluated using MAE, MSE, and R² score
Made predictions on test and new data
📈 Results
MAE: ~8174
MSE: ~101M
R² Score: ~0.998 (high accuracy)
🚀 Outcome

The model shows strong performance and effectively predicts house prices based on input features.
